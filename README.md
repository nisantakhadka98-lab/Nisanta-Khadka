## Project Overview
This project is part of **CSP451: Introduction to GitHub and Version Control**.  
It demonstrates fundamental Git and GitHub workflows, including:

- Creating and initializing a Git repository  
- Committing changes with meaningful messages  
- Branching, merging, and pull request workflows  
- Collaborating on GitHub with proper version control practices

The project uses basic **HTML, CSS, and JavaScript** to create a simple webpage with an About page.

---

## Features
- Fully initialized local Git repository with proper commit history  
- `.gitignore` to exclude unnecessary files  
- Branching workflow with feature branches and pull requests  
- Demonstration of backup and recovery using Git  
- Basic web page with personal information in `about.html`
